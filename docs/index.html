EEDE
GitHub

Expertのための、
研ぎ澄まされた描画環境。

Eedeは、32bit RGBAカラーをフルに活用し、1pxの狂いもなくドットを打つためのWindowsネイティブ・ドット絵エディタです。

Features

Eedeが提供する、3つのExcellent

静寂と没入

創作を邪魔しないミニマルなUI。ドッキング可能なパネルと自由なワークスペースにより、キャンバスとの対話にのみ集中できる「静かな空間」を提供します。

アニメーションの鼓動

直感的なシーケンサーとリアルタイムプレビュー。描いたドットが即座に命を宿す瞬間を、シームレスなワークフローで支えます。

技術を継承する

256色パレット（.act）やPC-98時代の画像形式（.ARV）の読み込みにも対応し、過去の貴重なドット絵資産を現代の環境に蘇らせます。

さあ、新しい環境で描こう。

Eedeは無料でオープンソース。あなたのピクセル・アートに妥協のない環境を。
